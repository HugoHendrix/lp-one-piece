# LP One Piece — Landing Page

Uma landing page temática de One Piece criada para praticar e demonstrar boas práticas em HTML, CSS e JavaScript.

## Objetivos
- Praticar HTML semântico e acessibilidade.
- Melhorar SEO e performance de páginas estáticas.
- Aplicar técnicas modernas de CSS: Flexbox, Grid, Custom Properties e Container Queries.
- Implementar interações acessíveis com JavaScript puro.

## Funcionalidades
- Layout responsivo e otimizado para dispositivos móveis.
- Estrutura semântica para melhorar SEO e navegação por leitores de tela.
- Estilos modernos com variáveis CSS e layout adaptativo.
- Interações simples e acessíveis (navegação, animações leves).

## Tecnologias
- HTML5 semântico
- CSS3 (Flexbox, Grid, Custom Properties, Container Queries)
- JavaScript (Vanilla)

## Estrutura do projeto
- index.html — página principal
- /assets — imagens, fontes e ícones
- /css — estilos
- /js — scripts

## Boas práticas / Notas
- Priorizar carregamento rápido de imagens (formatos otimizados e lazy loading).
- Verificar contraste e navegabilidade por teclado.
- Documentar alterações e manter CSS modular.
